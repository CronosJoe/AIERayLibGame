
Introduction to C# assessment game, Block Blocker, by Joseph Huskey.

Written against .NET core 3.1, in Visual studio 2019, using Raylib 3.1.5

To build the game, right click on the project in the solution explorer and click publish. Then select your desired folder you wish to build to and press publish.

Block blocker is a 2D defender type game built in ray lib, your goal is to stop the enemy blocks from moving down past the player. 

Controls are the left/right arrows for movement and space to fire. Alternatively you can use A/D for left/right movement.
